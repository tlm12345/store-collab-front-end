# 更新日志

## [2.0.0] - 2026-03-15

### 🎨 重大更新：全新设计系统

#### 设计理念
- 极简主义与沉浸式视觉相结合的现代风格
- 以中性灰白或深色模式为基底
- 高对比度的强调色突出"协作"功能按钮
- 悬浮式导航栏，确保浏览连贯性
- 平滑的微交互动画

#### 色彩系统
- 主色调：渐变紫蓝 (#667eea → #764ba2)
- 中性色：灰白色系，营造专业感
- 功能色：清晰的成功/错误状态

#### 页面重构

**登录页面** (`src/views/LoginView.vue`)
- 全新的左右分栏设计
- 左侧品牌展示区域，带动画效果
- 右侧简洁的登录表单
- 响应式设计，移动端优化
- 渐变背景，浮动装饰元素

**注册页面** (`src/views/RegisterView.vue`)
- 与登录页面保持一致的设计风格
- 圆形头像上传，悬停效果
- 多步骤表单，清晰的字段标签
- 实时字符计数
- 优化的错误提示

**个人信息页面** (`src/views/ProfileView.vue`)
- 全新的卡片式布局
- 渐变背景头像区域
- 信息网格展示
- 统计数据展示
- 响应式设计

**首页** (`src/views/HomeView.vue`)
- 全新的英雄区域设计
- 浮动卡片动画
- 特性网格展示
- CTA（号召性）区域
- 渐变背景动画

**应用导航** (`src/App.vue`)
- 悬浮式顶部导航栏
- 毛玻璃效果背景
- 桌面端/移动端自适应
- 折叠式移动端菜单
- 平滑的过渡动画

## 版本说明

### 版本号格式

遵循 [语义化版本 2.0.0](https://semver.org/lang/zh-CN/) 规范：

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能新增
- **修订号**：向下兼容的问题修正
