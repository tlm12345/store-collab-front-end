# Cloud Collab 设计系统

## 设计理念

Cloud Collab 采用**极简主义与沉浸式视觉相结合**的现代风格，为用户提供专业、高效且无干扰的云端存储与协同创作体验。

### 核心原则

1. **极简主义**：去除不必要的装饰元素，突出核心内容
2. **沉浸式体验**：全屏布局，最大化内容展示
3. **微交互**：平滑的动画和过渡效果，提升用户体验
4. **响应式设计**：完美适配各种设备尺寸

## 组件样式

### 按钮

#### 主按钮

```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
color: white;
padding: 0.875rem 2rem;
border-radius: 12px;
box-shadow: 0 4px 12px rgba(102, 126, 234, 0.4);
```

**悬停效果**：
- 向上移动 2px
- 阴影增大

#### 次要按钮

```css
background: white;
color: #667eea;
border: 2px solid #667eea;
```

### 输入框

```css
padding: 0.875rem 1rem;
border: 2px solid #e5e7eb;
border-radius: 12px;
background: #fafafa;
color: #1a1a1a;
```

**聚焦状态**：
- 边框颜色变为品牌色
- 背景变为白色
- 外发光效果

### 卡片

```css
padding: 2rem;
background: white;
border-radius: 16px;
box-shadow: 0 2px 8px rgba(0, 0, 0, 0.05);
```

**悬停效果**：
- 向上移动 8px
- 阴影增大

## 动画系统

### 过渡时间

| 名称 | 值 | 用途 |
|------|-----|------|
| 快速 | 0.2s | 颜色变化 |
| 标准 | 0.3s | 大多数交互 |
| 慢速 | 0.5s | 页面切换 |

### 缓动函数

```css
transition: all 0.3s ease;
```

### 关键动画

#### 浮动动画

```css
@keyframes float {
  0% { transform: translate(0, 0); }
  100% { transform: translate(50px, 50px); }
}
```

#### 卡片浮动

```css
@keyframes floatCard {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-20px); }
}
```

## 最佳实践

1. **保持一致性**：在整个应用中使用相同的颜色、字体和间距
2. **使用渐变**：按钮和重要元素使用品牌渐变色
3. **添加阴影**：卡片和按钮使用适当的阴影增强层次感
4. **圆角设计**：使用大圆角（16px+）营造现代感
5. **动画过渡**：所有交互都应该有平滑的过渡效果
6. **响应式优先**：移动优先设计，逐步增强桌面体验
